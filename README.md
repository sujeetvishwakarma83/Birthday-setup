# 🎂 Custom Birthday Message – Interactive Web App

A fun, lightweight, and fully responsive web application that lets users select their birthday and instantly see a personalized message — built using **HTML, CSS, and JavaScript**.

---

## 🚀 Live Demo

👉 https://sujeetvishwakarma83.github.io/Birthday-setup/

---

## ✨ Features

- 🎨 Clean UI with **frosted glass (glassmorphism) effect**
- ⚡ Instant message display (no page reload)
- 🖱️ Smooth **hover and transition effects**
- 📱 Fully **responsive design** (mobile, tablet, desktop)
- 📆 Dynamic **date formatting using JavaScript**

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling, layout, transitions  
- **JavaScript (Vanilla)** – DOM manipulation & date logic  

---

## 📱 Responsive Design

This project is built with a **mobile-first approach** using:

- Flexbox layout  
- Percentage-based widths  
- Max-width for proper scaling  

It works smoothly across all screen sizes.

---

## 💡 Use Cases

- 📘 Learning **JavaScript DOM & Date objects**
- 🎯 Practicing **CSS UI & transitions**
- 🧪 Beginner-friendly **mini project**
- 🧩 Base project for **event or reminder apps**

---

## 🧪 How It Works

1. Select your birthdate using the input field  
2. Click on **"Set Birthday"**  
3. Instantly see a formatted birthday message
